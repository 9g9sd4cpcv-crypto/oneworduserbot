# Telegram Userbot

A Pyrogram-based Telegram userbot with word chain integration.

## Features
- Tag System (`.tagall`, `.tagadmins`)
- Moderation (`.ban`, `.unban`, `.mute`, etc.)
- Word Chain with @on9wordchainbot
- Auto-welcome

## Setup

1. Get API credentials from [my.telegram.org](https://my.telegram.org)
2. Generate string session
3. Set environment variables
4. Deploy

## Commands
| Command | Description |
|---------|-------------|
| `.tagall` | Tag all users |
| `.tagadmins` | Tag admins |
| `.ban` | Ban user |
| `.wcstart` | Start word chain |
